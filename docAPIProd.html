Documentação da API REST - Produtos

1. Listar Todos os Produtos

Método: GET

Rota: /produtos

Retorna uma lista de todos os produtos cadastrados no banco de dados.

Exemplo de resposta:
[
  {
    "id": 1,
    "nome": "Produto A",
    "preco": 25.99
  },
  {
    "id": 2,
    "nome": "Produto B",
    "preco": 35.50
  }
]

2. Listar um Produto Específico

Método: GET

Rota: /produtos/<int:produto_id>

Retorna os dados de um produto específico pelo seu ID.

Exemplo de resposta:
{
  "id": 1,
  "nome": "Produto A",
  "preco": 25.99
}

3. Criar um Novo Produto

Método: POST

Rota: /produtos

Cria um novo produto com base nos dados fornecidos no corpo da requisição.

Exemplo de corpo da requisição (JSON):
{
  "nome": "Produto C",
  "preco": 45.75
}

Exemplo de resposta:
{
  "id": 3,
  "nome": "Produto C",
  "preco": 45.75
}

4. Atualizar um Produto

Método: PUT

Rota: /produtos/<int:produto_id>

Atualiza os dados de um produto específico pelo seu ID.

Exemplo de corpo da requisição (JSON):
{
  "nome": "Produto A Atualizado",
  "preco": 30.00
}

Exemplo de resposta:
{
  "id": 1,
  "nome": "Produto A Atualizado",
  "preco": 30.00
}

5. Deletar um Produto

Método: DELETE

Rota: /produtos/<int:produto_id>

Remove um produto específico do banco de dados pelo seu ID.

Exemplo de resposta:
{
  "mensagem": "Produto excluído com sucesso"
}
